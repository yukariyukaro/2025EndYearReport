# 项目启动指南

## 在浏览器中查看项目效果的步骤

### 第一步：安装依赖

在项目根目录打开终端（PowerShell 或 CMD），运行：

```bash
npm install
```

或者如果你使用 pnpm：

```bash
pnpm install
```

这会安装所有必需的依赖包（Next.js、React 等）。

### 第二步：启动开发服务器

安装完成后，运行：

```bash
npm run dev
```

你会看到类似这样的输出：

```
▲ Next.js 16.0.6
- Local:        http://localhost:3000
- Ready in 2.3s
```

### 第三步：在浏览器中打开

1. 打开浏览器（Chrome、Edge、Firefox 等）
2. 访问：`http://localhost:3000`
3. 你应该能看到你的年度报告页面了！

### 常见问题

**Q: 端口 3000 被占用怎么办？**
A: Next.js 会自动尝试下一个可用端口（如 3001、3002），查看终端输出中的实际地址。

**Q: 想从手机访问测试移动端效果？**
A: 使用 `npm run dev:host` 启动，然后：
- 确保手机和电脑在同一 WiFi 网络
- 在手机浏览器访问：`http://你的电脑IP:3000`
- 查看终端输出中的网络地址

**Q: 看到错误信息？**
A: 
- 确保已运行 `npm install`
- 检查 Node.js 版本（需要 18+）：`node -v`
- 查看终端中的错误详情

### 停止服务器

在终端中按 `Ctrl + C` 停止开发服务器。

---

**提示**：开发模式下，修改代码后页面会自动刷新，无需手动刷新浏览器！
